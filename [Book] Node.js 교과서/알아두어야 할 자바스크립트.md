자바스크립트는 2015년을 기점으로 문법에 큰 변화가 생겼다. 그리고 2015년 이후로 매년 문법이 바뀌고 있는데, 이를 통틀어서 ES2015+ 라고 한다.

### const, let

const와 let이 var을 대체한다.
var은 함수 스코프지만, const와 let은 블록 스코프다.
const는 변수값 재선언이 불가능하다.

### 템플릿 문자열

백틱을 쓸 수 있다. 문자열 안에 변수를 쉽게 넣을 수 있다.

### 객체 리터럴

```JavaScript
const newObject = {
  sayJs() {
    console.log('JS');
  },
  sayNode,
  [es + 6]: 'Fantastic',
};

```
